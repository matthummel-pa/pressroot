directory verbatim (marketplace reviewers expect the license file itself,
not only this summary). The Google Fonts collection feature
(`app/google-fonts-collection.php`) downloads additional families at the
site owner's request; those are fetched from Google Fonts at runtime and are
not redistributed with the theme.
